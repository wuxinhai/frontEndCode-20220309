梅西职业生涯已造1001球！ 夺金靴追平两大传奇，在巴萨踢到36岁？

本赛季西甲联赛已经结束，7月21日出版的《世界体育报》指出，梅西又创造了一项纪录： 职业生涯的制造进球数已经超过1000个!

西甲最后1轮，巴萨5-0大胜阿拉维斯，梅西贡献2球1助攻。今日出版的《世界体育报》在头版做出统计：出道至今，梅球王已经为巴萨和阿根廷国家队录得703球298次助攻，参与制造1001粒进球，突破千球里程碑！

此外，梅西第7次夺得西甲金靴奖，放眼欧洲主流联赛，这样的获奖次数，已经追平盖德-穆勒（德甲）和尤西比奥（葡超）保持的纪录，下赛季有望实现超越！ 不过，巴萨球迷最关心的还是梅西的未来。而零点电台的资深巴萨跟队记者阿尔弗雷多-马丁内斯的一句话，也许会令萨迷放心，他表示：“我认为梅西会在巴萨踢到36岁。” 如今梅西33岁，踢到36岁意味着再踢三年，这么看，续约是没有什么问题的了！
